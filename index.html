dateutil-tzdata

Time zone data for dateutil

Mirror of the Olson timezone database with python-dateutil-accessible data and metadata and pre-built dateutil zoneinfo files.

The main site for these data files can be found at the IANA's website. The archive of releases is here (FTP). More information can be found at the code repository. I have verified that all files from version 2012j onwards have been signed using key ID 0x62AA7E34. The built zoneinfo files are signed with my own key, 0xD964BEFB.

To use these files with dateutil, run updatezinfo.py with the appropriate arguments to build the zoneinfo database - this requires zic. For convenience, a zoneinfo metadata file is provided alongside each release, which can be used as an input to updatezinfo.py. Additionally, prebuilt zoneinfo files are provided for each release, and can be placed directly in your dateutil/zoneinfo file (it must be named dateutil-zoneinfo.tar.gz).
